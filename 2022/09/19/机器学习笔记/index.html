basic knowledge

what is the machine learning

人工智能的核心，计算机拥有智能的根本途径。是从数据中发现复杂规律，并用规律对未知状况、未来时刻进行预测和判定的方法。机器学习理论上是设计和分析一些让计算机自动学习的算法。

过程大致总结为：训练数据->建立模型->预测未知属性

classification of machine learing

  • unsupervised learing 无监督学习：数据并不被特别标识，计算机自行学习分析数据内部的规律、特征，进而得出结果。

    • 典型例子：聚类算法、GAN(生成对抗算法)

  • supervised learing 监督学习：在建立预测模型的过程中将预测结果与训练数据的实际结果进行比较，不断调整模型，知道模型的预测结果达到预期的准确率。

    • 典型例子：分类和回归任务、决策树、贝叶斯模型、支持向量机、深度学习

  • semi-surpervised learning 半监督学习：介于上述二者之间，部分输入数据被标识，部分没有，没有标识的数据数量常常远大于有标识的数据

  • reinforcement learning 强化学习

basic terminology

  • Classification 分类： 使用计算机学习出的模型进行预测，得到的是离散值；输出的是物体的类别；是定性的；目的是为了寻找决策边界，把类别边界找到即可

    • 线性分类

      • 二分类：只涉及两个类别的分类任务，一个为正类(positiv class)，另一种为负类(negative class)。例如是猫和不是猫两类

      • 多分类 (multi-class classification)：涉及多个类别的分类任务

    • 非线性分类

      • 例如分类在圆环上的数据集，无法用直线进行分割

  • Regression 回归： 使用计算机学习出的模型进行预测，得到的是连续值；输出物体的值；是定量的；目的是为了找到最优拟合；不仅实现了分类，还给出对真实值的逼近预测

单看这个词，regression.re前缀表示的是恢复(recover),重复(repeat)的意思，sion尾缀把动词变成名词，gress的意思是行走。想象这样一个场景，一堆看似无规律的数据在你的图像上肆无忌惮的乱走，我们要做的是找出其中的规律模型，把他们行走的趋势和轨迹“重新组合起来”。

原文链接：https://www.zhihu.com/question/30123729/answer/46958971

回归分析是指一种预测性的建模技术，主要是研究自变量和因变量的关系。通常使用线/曲线来拟合数据点，然后研究如何使曲线到数据点的距离差异最小。
